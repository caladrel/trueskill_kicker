    About

    kicker-league is an open-source table football tracking / rating system.

    It uses a rating algorithm developed by the research division of a well-known software corporation from Redmond. The skill of the player is quantified from Bayesian inference. kicker-league uses a ready-made implementation of the algorithm in python, which is openly available

    Get the Code

    kicker-league is currently developed on GitHub.

    You can find the repository at https://github.com/caladrel/trueskill_kicker
